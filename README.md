# Bean & Brew Coffeehouse

A modern, responsive multi-page web application built for a premium local coffee shop. This project features a completely responsive layout, uniform navigation across pages, a dark mode toggle, and a dynamic interactive contact form.

## Technology Stack
Built using semantic HTML5, Cascading Style Sheets (CSS3), dynamic client-side JavaScript, and the Bootstrap v5 framework.

## Project Structure
* `index.html` - Home Page
* `about.html` - Our Story
* `menu.html` - Curated Menu
* `gallery.html` - Space & Crafts Showcase
* `faq.html` - Frequently Asked Questions
* `blog.html` - Bean & Brew Journal
* `contact.html` - Contact Form
* `styles.css` - Custom Global Stylesheets
* `script.js` - Client-side Interactive Logic
